SNAP eligibility for any UPC, in any state, in one API call.

2026 SNAP restrictions differ across 22 states and keep changing. SnapScan keeps your EBT flags accurate — pass a UPC and a state code, get back eligible: true/false with reasons. Cache-backed, audit-logged, built for checkout latency.

MethodPathPurpose
